## Tech Stack

- **Frontend:** Next.js (App Router), Tailwind CSS, Shadcn/UI
- **Backend:** Supabase (PostgreSQL, Storage)
- **AI:** Google Gemini 3.0 Flash
- **Payments:** Stripe
